Приложение Stegos объединяет кошелек и мессенджер
Создание рассылок токенов, на которые может претендовать любой, у кого есть QR-код. Красные пакеты - это сюрприз (в китайской традиции), который вы можете отправить кому угодно
Безопасная мобильная виртуальная машина для запуска приложений, написанных на HTML, CSS и JavaScript. Простой API для обмена сообщениями, полностью независимый от базового блокчейна
Запусти ноду на своем смартфоне и зарабатывай токены, помогая поддерживать сеть Stegos
